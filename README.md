# Portfolio

Personal developer portfolio showcasing my projects, skills, and experience.

## Built With

- **React 19** with Vite
- **Motion** for scroll-reveal animations
- **CSS Modules** for scoped styling
- Fully responsive design

## Features

- Smooth scroll navigation with active section highlighting
- Animated section reveals on scroll
- Project cards with live demo links and GitHub repos
- Mobile-friendly phone screenshot previews for mobile app projects
- Timeline-based experience section

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) to view it locally.

## Build

```bash
npm run build
```

Output goes to the `dist/` directory, ready for static hosting.
